Planificador de Tareas

Sistema completo de planificación y gestión de tareas desarrollado con Flask (Python).

Características Principales

  • ✅ Creación de Tareas: Crea y gestiona tareas con descripción completa
  • 👤 Asignación de Responsables: Asigna personas responsables a cada tarea
  • 🏢 Centros de Responsabilidad: Organiza tareas por centros de responsabilidad
  • 📅 Gestión de Plazos: Define y controla plazos de entrega
  • 📎 Medios de Verificación: Adjunta documentos y archivos de soporte
  • 💼 Recursos: Especifica recursos necesarios para cada tarea
  • 📝 Observaciones: Añade observaciones y notas adicionales
  • 🔄 Estados de Tareas: Controla el estado (Sin Ejecutar, En Ejecución, Pendiente de, Completada, Validada)
  • 📊 Tablero de Control: Visualiza estadísticas con gráficos y tablas
  • 📧 Alertas por Correo: Recibe notificaciones de tareas por vencer
  • 📄 Informes: Genera informes completos del estado del arte

Estados de Tareas

  • Sin Ejecutar - Tarea aún no iniciada
  • En Ejecución - Tarea en progreso
  • Pendiente de - Esperando validación o aprobación
  • Completada - Tarea finalizada
  • Validada - Tarea verificada y aprobada

Tecnologías Utilizadas

  • Backend: Python 3, Flask
  • Frontend: HTML5, CSS3, JavaScript
  • Gráficos: Chart.js
  • Persistencia: JSON
  • Notificaciones: SMTP (correo electrónico)

Configuración de Alertas por Correo

Para activar las alertas por correo electrónico, configura las siguientes variables de entorno:

EMAIL_HOST=smtp.tu_proveedor.com
EMAIL_PORT=587
EMAIL_USER=tu_correo@dominio.com
EMAIL_PASSWORD=tu_contraseña
EMAIL_FROM=tu_correo@dominio.com
EMAIL_TO=destinatario@dominio.com

Nota: Las alertas se verifican automáticamente cada 60 minutos para tareas que vencen en los próximos 2 días.
